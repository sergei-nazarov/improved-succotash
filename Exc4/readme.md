| Роль           | Права роли                                                                                               | Группы пользователей                   |
|----------------|---------------------------------------------------------------------------------------------------------|----------------------------------------|
| `viewer`       | Чтение ресурсов кластера: `get`, `list`, `watch` для всех ресурсов (например, `pods`, `services`)       | Группа пользователей с правом просмотра |
| `configurator` | Чтение и изменение конфигурации кластера: `get`, `list`, `watch`, `create`, `update`, `delete` для ресурсов, таких как `configmaps`, `deployments` | Группа пользователей с правом настройки кластера |
| `admin`        | Привилегированный доступ, включая просмотр секретов: `get`, `list`, `watch` для секретов и всех других ресурсов кластера | Привилегированная группа пользователей |
